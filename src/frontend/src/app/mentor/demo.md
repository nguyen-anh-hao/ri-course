# Hướng dẫn cho Mentor

Chào mừng bạn đến với hệ thống mentor. Dưới đây là các bước bạn cần thực hiện:

1. Đăng nhập vào hệ thống.
2. Hoàn thành hồ sơ cá nhân.
3. Bắt đầu hướng dẫn học viên.
